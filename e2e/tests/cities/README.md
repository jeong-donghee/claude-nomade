# 도시 테스트

이 디렉토리는 도시 관련 E2E 테스트를 포함합니다.

## 📋 테스트 파일 목록

- `city-list.spec.ts` - 도시 목록 조회 테스트
- `city-detail.spec.ts` - 도시 상세 페이지 테스트
- `city-search.spec.ts` - 도시 검색 기능 테스트
- `city-filter.spec.ts` - 카테고리 필터링 테스트
- `city-sort.spec.ts` - 정렬 기능 테스트

## 🎯 테스트 시나리오

### city-list.spec.ts
- [ ] 도시 목록 페이지가 정상적으로 렌더링
- [ ] 모든 도시 카드가 표시됨
- [ ] 도시 카드에 기본 정보 표시 (이름, 평점, 생활비 등)
- [ ] 페이지네이션 또는 무한 스크롤 동작

### city-detail.spec.ts
- [ ] 도시 상세 페이지가 정상적으로 렌더링
- [ ] 도시 상세 정보 표시 (장점, 단점, 추천 장소 등)
- [ ] 좋아요 버튼 표시
- [ ] 뒤로가기 동작

### city-search.spec.ts
- [ ] 한글 검색 기능 동작
- [ ] 영문 검색 기능 동작
- [ ] 검색 결과 실시간 업데이트
- [ ] 검색어 초기화 버튼 동작
- [ ] 검색 결과 없을 시 메시지 표시

### city-filter.spec.ts
- [ ] 카테고리 필터 버튼 클릭 시 필터링
- [ ] 전체 카테고리 선택 시 모든 도시 표시
- [ ] 특정 카테고리 선택 시 해당 도시만 표시
- [ ] 필터링 결과 실시간 업데이트

### city-sort.spec.ts
- [ ] 인기순 정렬 동작
- [ ] 평점순 정렬 동작
- [ ] 생활비 낮은순 정렬 동작
- [ ] 좋아요순 정렬 동작
- [ ] 최신순 정렬 동작

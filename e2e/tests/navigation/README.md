# 네비게이션 테스트

이 디렉토리는 네비게이션 관련 E2E 테스트를 포함합니다.

## 📋 테스트 파일 목록

- `navbar.spec.ts` - 네비게이션 바 테스트
- `footer.spec.ts` - 푸터 테스트
- `mobile-menu.spec.ts` - 모바일 메뉴 테스트

## 🎯 테스트 시나리오

### navbar.spec.ts
- [ ] 네비게이션 바가 모든 페이지에 표시됨
- [ ] 로고 클릭 시 홈페이지로 이동
- [ ] 홈 링크 클릭 시 홈페이지로 이동
- [ ] 도시찾기 링크 클릭 시 도시 목록 페이지로 이동
- [ ] 로그인 상태일 때 사용자 이메일 표시
- [ ] 로그인 상태일 때 로그아웃 버튼 표시
- [ ] 비로그인 상태일 때 로그인/회원가입 버튼 표시

### footer.spec.ts
- [ ] 푸터가 모든 페이지 하단에 표시됨
- [ ] 푸터 링크들이 정상 동작
- [ ] 저작권 정보 표시

### mobile-menu.spec.ts
- [ ] 모바일 화면에서 햄버거 메뉴 아이콘 표시
- [ ] 햄버거 메뉴 클릭 시 메뉴 열림
- [ ] X 아이콘 클릭 시 메뉴 닫힘
- [ ] 메뉴 링크 클릭 시 해당 페이지로 이동하고 메뉴 닫힘
- [ ] 메뉴 외부 클릭 시 메뉴 닫힘

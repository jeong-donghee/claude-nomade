# 좋아요 테스트

이 디렉토리는 좋아요 기능 관련 E2E 테스트를 포함합니다.

## 📋 테스트 파일 목록

- `like-toggle.spec.ts` - 좋아요 토글 기능 테스트
- `like-count.spec.ts` - 좋아요 수 표시 테스트
- `like-persistence.spec.ts` - 좋아요 상태 유지 테스트

## 🎯 테스트 시나리오

### like-toggle.spec.ts
- [ ] 로그인 상태에서 좋아요 버튼 클릭 시 좋아요 추가
- [ ] 좋아요한 도시 다시 클릭 시 좋아요 제거
- [ ] 비로그인 상태에서 좋아요 클릭 시 로그인 안내
- [ ] Optimistic UI 동작 확인 (즉시 UI 업데이트)
- [ ] 하트 아이콘 색상 변경 (♡ ↔ ♥)

### like-count.spec.ts
- [ ] 도시 카드에 좋아요 수 표시
- [ ] 좋아요 추가 시 카운트 증가
- [ ] 좋아요 제거 시 카운트 감소
- [ ] 여러 사용자의 좋아요 수 집계 정확성

### like-persistence.spec.ts
- [ ] 페이지 새로고침 후 좋아요 상태 유지
- [ ] 로그아웃 후 재로그인 시 좋아요 상태 유지
- [ ] 도시 목록에서 상세 페이지로 이동 시 상태 일관성
- [ ] 브라우저 재시작 후 상태 유지

# 통합 시나리오 테스트

이 디렉토리는 여러 기능을 결합한 통합 시나리오 E2E 테스트를 포함합니다.

## 📋 테스트 파일 목록

- `user-journey.spec.ts` - 사용자 여정 전체 테스트
- `like-flow.spec.ts` - 좋아요 전체 플로우 테스트

## 🎯 테스트 시나리오

### user-journey.spec.ts
완전한 사용자 여정을 시뮬레이션합니다:

1. [ ] 홈페이지 접속
2. [ ] 도시 목록 페이지로 이동
3. [ ] 카테고리 필터링 사용
4. [ ] 도시 검색 사용
5. [ ] 도시 상세 페이지 조회
6. [ ] 회원가입 시도
7. [ ] 이메일 인증 (모의)
8. [ ] 로그인
9. [ ] 도시 좋아요 추가
10. [ ] 도시 목록에서 좋아요 상태 확인
11. [ ] 로그아웃
12. [ ] 다시 로그인하여 좋아요 상태 유지 확인

### like-flow.spec.ts
좋아요 기능의 전체 플로우를 테스트합니다:

1. [ ] 비로그인 상태에서 도시 조회
2. [ ] 좋아요 버튼 클릭 시 로그인 안내
3. [ ] 로그인 페이지로 이동
4. [ ] 로그인 성공
5. [ ] 이전 페이지로 돌아가기
6. [ ] 좋아요 버튼 클릭하여 추가
7. [ ] 좋아요 수 증가 확인
8. [ ] 다른 도시도 좋아요 추가
9. [ ] 좋아요한 도시 필터링 (향후 기능)
10. [ ] 좋아요 취소
11. [ ] 좋아요 수 감소 확인
